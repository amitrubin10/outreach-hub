# 🎯 מרכז שיווק בקבוצות (Outreach Hub)

כלי אישי **רב-פרויקטי** לשיווק בקבוצות פייסבוק — איתור פוסטים רלוונטיים, תגובות מוכנות,
מעקב לידים, וסטודיו ליצירת תוכן. עצמאי לחלוטין: **קובץ HTML יחיד, בלי שרת, בלי התקנה.**

## איך מפעילים
לחיצה כפולה על **`index.html`** — נפתח בדפדפן. זהו.

## מה יש בפנים
- **📁 פרויקטים** — כל מוצר (VeriBayit, ובעתיד אחרים) עם קבוצות/תגובות/לידים משלו.
- **👥 קבוצות** — כפתור "פתח קבוצה" + כפתורי **חיפוש בתוך הקבוצה** לפי מילות מפתח + צ'קליסט יומי.
- **📌 פוסטים** — מעקב מי הגבת לו ומי לא.
- **👤 לידים** — מיני-CRM עם סטטוסים ומעקב חוזר.
- **💬 תגובות** — תבניות מוכנות עם כפתור "העתק".
- **✍️ סטודיו פוסטים** — יצירת תוכן עם רעיונות-פתיחה.

## נתונים וגיבוי
- הכל נשמר ב-`localStorage` של **הדפדפן הזה** (לא בענן).
- **חשוב:** מדי פעם ייצא גיבוי (⚙️ הגדרות → ייצוא) כדי לא לאבד מידע / להעביר למכשיר אחר.

## הערה אתית
לשימוש **ידני בלבד** — בלי בוטים שסורקים/מגיבים אוטומטית (נגד תנאי פייסבוק + חוסם חשבונות).
הכלי רק **מארגן ומזרז** את העבודה הידנית. תמיד להגיב אישית ואמפתית.
